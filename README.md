# SimuSolve

SimuSolve is a React + Vite workspace for turning physics problems into
interactive step-by-step explanations with a live 3D trajectory simulation.

## Stack

- React 19 + Vite 8
- Tailwind CSS 4
- Three.js with `@react-three/fiber` and `@react-three/drei`
- Recharts
- Matter.js
- Axios

## Project Structure

```text
src/
  components/
  simulations/
  hooks/
  utils/
  pages/
```

## Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm run build
npm run lint
```
